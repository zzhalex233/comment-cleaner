# Comment Cleaner

一个用于 批量清理代码注释 的 Windows 小工具。  

## 功能

- 清理行注释  块注释  
- 预览模式（不写回文件）
- Unified Diff 预览
- Side-by-side 行对比
- 文件夹模式  单文件模式
- 按后缀过滤
- 写回前自动备份

## 支持环境

- Windows 10  11
- .NET 8
https://dotnet.microsoft.com/zh-cn/download/dotnet/8.0

## 使用

1. 从 Releases 下载
2. 双击运行
